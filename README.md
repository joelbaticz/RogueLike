# RogueLike
A high performant 2D/3D Game Engine created for a rouge like game. With asset management (loading, caching, reference counting), Polygon and Sprite handling, Software Rasterization and Hardware Accelerated sprite blitting, fast text and texture rendering capabilities, using C++ and the SDL library.
